# Fashion Sentiment Analysis

This repository contains a dataset of 1,200 fashion-related tweets collected using the Twitter API v2 and processed using Python.

## Files Included

- `fashion_sentiment_dataset.csv` — Collected dataset with tweet text, metadata, and sentiment labels.
- `data_collection.py` — Python script used to collect and process the tweets.
- `Twitter_Data_Methodology.docx` — Detailed methodology explaining the source, tools used, and ethical considerations.

## Tools Used

- Python
- Tweepy (Twitter API library)
- TextBlob (for sentiment analysis)
- Pandas (for data handling)

## Ethical Notice

- Only public tweets were collected.
- No personal identifiers (e.g., usernames, email, location) were included.
- Dataset was collected solely for academic research, following Twitter's Developer Policy.
